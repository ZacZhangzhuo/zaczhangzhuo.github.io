刚学Visual Studio GH Plagin时练习的一个生长的面，看上去很像一个大脑，就随手做了这样一个海报。



Grasshopper的Plugin主要是用VS（Visual Studio)做的。比较专业，相比之下Plugin也比直接在GH上的C#运算电池起来更快速。对于使用者也更加友好。关键是VR是专业的编程软件，在纠错、提醒、Debug上十分便捷，对于复杂的编程优势明显，所以应该了解涉及下。



首先要下载templates，也就是一种集成开发环境IDE（Integrated Development Enviroment)，这个是使用VS做电池必须的。下载好了VS后，可以在

（https://marketplace.visualstudio.com/items?itemName=McNeel.GrasshopperAssemblyforv6）

基础的入门大致就这些，真正学到Plugin的方法并且走向熟练，我个人认为还是要多看视频，多练习。毕竟编程本就是一门独立的科目，里面包含类型、结构、属性、继承、数据结构等很多方面，需要的不仅是编程手法，还要数学思维等。并且不同类型的建筑师相信对这方面的深度要求也不一样。所以我在搬运知识的同时也希望我们一起交流进步，取长补短。
